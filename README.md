# 📖 Descrição

O **Site Interativo** é uma aplicação web desenvolvida com o objetivo de demonstrar a criação de interfaces dinâmicas utilizando tecnologias fundamentais do desenvolvimento front-end.

A aplicação combina **estrutura HTML**, **estilização CSS** e **lógica em JavaScript** para criar uma experiência interativa no navegador. O projeto foi desenvolvido com foco em explorar conceitos essenciais de construção de interfaces web, manipulação de elementos da página e interação com o usuário.

Aplicações desse tipo são comuns em páginas institucionais, portfólios, landing pages e sistemas web que precisam oferecer **interação dinâmica e feedback visual ao usuário**.

---

# 🚀 Preview do Projeto

Fluxo simplificado do funcionamento da interface:

```
Usuário acessa a página
        ↓
Interface é carregada no navegador
        ↓
Elementos interativos respondem às ações do usuário
        ↓
JavaScript manipula os elementos da página
        ↓
A interface é atualizada dinamicamente
```

O projeto demonstra como **eventos do usuário (cliques, interações e navegação)** podem alterar dinamicamente o conteúdo exibido na página.

---

# 🧠 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais da web:

* **HTML5** — estrutura e organização do conteúdo da página
* **CSS3** — estilização, layout e responsividade
* **JavaScript (ES6)** — lógica de interação e manipulação da interface

Essas tecnologias permitem que o projeto funcione **diretamente no navegador**, sem necessidade de frameworks ou dependências externas.
